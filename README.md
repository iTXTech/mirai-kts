# Mirai Kts

**强大的 `Mirai Kotlin Script` 插件运行时**

使用`Kotlin Script`编写 [Mirai](https://github.com/mamoe/mirai) 插件，支持与所有`Mirai API`直接交互，现仅支持`OpenJDK`环境。

## 使用须知

**所有基于`Mirai Kts`的插件必须遵循`AGPL-v3`协议开放源代码，详见 [协议文本](LICENSE) 。**

**API可能随时变动，请保持更新！**
